# 常用命令：
export 环境变量名=value ： 环境变量设置，注意 = 两边不能有空格，而且该设置仅在当前打开的 terminal 中有效，要想让这个环境变量在全局中起作用，需要将该指令写入~/.bashrc.sh 文件里面，在 source ~/.bashrc.sh即可

unset 环境变量名 ： 删除该环境变量

env | grep 环境变量名： 输出环境变量名（或含有该环境变量名字符串的其它环境变量名）对应的值

echo $环境变量名 ： 这个$不可省略，打印该环境变量名对应的值


### shell中的通配符：
1. \* 匹配0个或多个任意字符
2. ? 匹配一个任意字符
3. \[若干字符\] 匹配方括号中任意一个字符的一次出现

举例：例如当前目录下有t3.sh文件，此时要找到列出该文件可以：
1. ls \*
2. ls \[abcdeft\]?.sh
3. ls t\[0-2\].sh  此时0-2表示可取0，1，2这三个数字


### 命令代换

date：打印当前时间

直接 DATE=date然后echo $DATE无法打印出当前时间，因为并没有执行date这个命令并赋值个DATE，因此需要使用命令代换

DATA=$(date) 或者 DATA=\`date
